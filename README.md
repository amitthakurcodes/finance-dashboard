# 💰 Finance Dashboard

A modern personal finance dashboard to track income, expenses, and categories with analytics and charts.

## 🚀 Features
- Add & manage transactions
- Track income & expenses
- Category-based filtering
- Charts & analytics
- Export CSV

## ⚙️ Tech Stack
- React (Vite)
- Tailwind CSS
- Supabase

## 📦 Run Locally
npm install
npm run dev

## 🔐 Environment Variables
Create a .env file and add:

VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key
